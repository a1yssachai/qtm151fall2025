c) For-loops + Plotting

  • Create an array of x-values with 100 evenly spaced points between $0$ and $2\pi$
    using np.linspace(). Hint: type help(np.linspace) in a separate cell to see how it is used

  • Create a vector named vec_y1 where each element is computed from the formula
    $ y_k = \sin(x_k)$. (Here $x_k$ is the $k$-th element of the array you created above, and $y_k$ is the $k$-th element of vec_y1)

  • Create a vector vec_y2 where each element is computed from the formula
    $ y_k = \cos(x_k)$.

  • Create a list list_vectors with elements vec_y1 and vec_y2. (So a list with two components, both being numpy arrays)

  • Create a for-loop that goes through each element of list_vectors and plots each
    list of $y$ values on the vertical axis against $x$ on the horizontal axis.

Note: The final output should be a 2-dimensional plot with the values from vec_y1 and vec_y2 on the vertical axis, and the $x$ values on the horizontal axis. Remember to label the axes and add a title to the plot.

In[9]:
# Write your answer here:
import matplotlib.pyplot as plt

x_vals = np.linspace(start = 0, stop = 2*np.pi, num = 100)

vec_y1 = np.sin(x_vals)
vec_y2 = np.cos(x_vals)

list_vectors = [vec_y1, vec_y2]

for vec in list_vectors:
    plt.plot(x_vals, vec)

plt.xlabel("$x$")
plt.ylabel("$y$")
plt.title("Graph of Sine and Cosine Functions for $0 \leq x \leq 2\pi$")
plt.show()
